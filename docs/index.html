Harika’s Free Dictionary API

A simple, free API that returns English word definitions. No signup required.

Live: https://dictionary-api-7hmy.onrender.com

Quick Start

Call the /define endpoint with a query parameter word:

GET https://dictionary-api-7hmy.onrender.com/define?word=apple

Example (curl)

curl "https://dictionary-api-7hmy.onrender.com/define?word=apple"

Example response

{
  "word": "apple",
  "partOfSpeech": "noun",
  "definition": "A common, round fruit produced by the tree Malus domestica..."
}

Parameters

  • word (required) — the word to look up, e.g. ?word=hello

HTTP Statuses

  • 200 — success (JSON body)
  • 400 — bad request (missing ?word=)
  • 404 — word not found
  • 500 — server error
